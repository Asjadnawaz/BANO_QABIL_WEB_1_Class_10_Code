# 🌐 Web Development with AI – Class 10

This repository contains my Class 10 practice work from the Web Development with AI course.

In this session, I focused on improving CSS skills by working with background images and learning Flexbox for better layout design.

## 📚 What I Learned
- Using `background-image` in CSS
- Background properties: `background-position`, `background-repeat`, `background-size`
- CSS Flexbox basics for layout design
- Aligning items using `justify-content` and `align-items`
- Removing default list styles using `list-style`

## 🎯 Practice Work
- Created a navbar using CSS Flexbox
- Aligned menu items in a horizontal row
- Positioned logo and navigation properly
- Applied background image to a webpage
- Improved layout structure and design consistency

## 🚀 Goal
To learn modern CSS layout techniques and build better, responsive website structures using Flexbox and background styling.
